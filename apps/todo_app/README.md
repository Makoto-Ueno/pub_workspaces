# Todo App

タスクを管理できるTODOアプリです。

## 機能

- **タスク追加**: タイトルと説明を入力してタスクを作成
- **タスク編集**: 既存のタスクを編集
- **完了管理**: チェックボックスで完了/未完了を切り替え
- **タスク削除**: 不要なタスクを削除
- **統計表示**: 完了・未完了の件数を表示
- **バリデーション**: タイトルの入力チェック

## 使用している共通パッケージ

- `core_utils`: Validator（入力チェック）, Formatter（日時・件数表示）
- `ui_components`: CustomButton, InfoCard（統計カード）

## 実行方法

```bash
cd apps/todo_app
flutter run
```

## 特徴

入力バリデーションにValidatorを活用し、適切なエラーメッセージを表示します。
統計情報をInfoCardで見やすく表示しています。
